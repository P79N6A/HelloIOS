
# CocoaPods

## 安装
**修改源**
```
//删除源
$sudo gem sources -r https://rubygems.org/

//添加源
$sudo gem sources -a https://gems.ruby-china.com

//查看源
$ gem sources -l

//安装 cocoapods
$ sudo gem install cocoapods

//查看pod版本
$ pod --version

//
```
## pod常用命令
```
//查看pod版本
$ pod --version

//安装pod
$ pod setup

//查询是否支持某个框架
$ pod search AFNetworking
```
##项目中应用
1.  `pod init`
1.  `pod install`
